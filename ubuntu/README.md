## Shortcut

- Settings > Keyboard
- Keyboard Shortcuts > Custom Shortcuts
- Add ablert shortcut
  - Name: `Show Albert`
  - Command: `albert show`
  - Shortcut: `Ctrl` + `Space`
- Add copyq shortcut
  - Name: `Show CopyQ`
  - Command: `copyq show`
  - Shortcut: `Ctrl` + `Shift` + `V`
