# Usage

### zsh config

Please place zsh config files in this directory.

```
# ~/.zsh.d/common/aliases.zsh

alias foo='echo "foo"'
````

### secret zsh config

Directory not added to git other than common.

```
# ~/.zsh.d/secret.zsh

export SECRET=XXX
````

# Linux


```
# ~/.zsh.d/linux.zsh

export pbcopy="xclip -selection clipboard"

export VISUAL=vim
```
